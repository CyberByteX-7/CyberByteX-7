## Featured projects

### 🌍 Earth Digital Twin — Multi-Hazard Disaster Prediction System
> Real-time AI framework for flood, landslide, and food insecurity prediction using a Digital Twin architecture

- CNN-based spatial analysis on satellite imagery for flood and landslide susceptibility mapping
- LSTM/RNN time-series forecasting for rainfall patterns and climate variability
- Live data ingestion via Tomorrow.io Weather API with real-time Digital Twin synchronization
- Interactive risk dashboard with geospatial heatmaps and automated early-warning alerts
- **Stack:** Python · PyTorch · TensorFlow · Scikit-learn · Tomorrow.io API · Sentinel/MODIS Satellite Data

---

### 🤖 Agentic Coding Framework — Self-Hosted LLM Orchestration
> Modular agentic coding skill system built on local inference — a self-hosted alternative to cloud coding agents

- Designed and deployed end-to-end inference pipelines on RTX 4090 using Ollama
- Evaluated and benchmarked DeepSeek-V3, Qwen 2.5 Coder, and GLM-4 series for agentic coding tasks
- Built custom tool-use and skill orchestration layers on top of local model inference
- Optimized quantization tradeoffs (Q4/Q8) for performance under consumer hardware constraints
- **Stack:** Python · Ollama · DeepSeek-V3 · Qwen 2.5 Coder · GLM-4 · Custom toolchains

---

### 🌊 Flood Probability Mapping — Deep Learning on Remote Sensing Data
> Per-pixel flood probability prediction using a ResNet-inspired deep learning architecture

- Implemented encoder-decoder architecture with skip connections for spatial segmentation
- Applied NDWI (Normalized Difference Water Index) for water body detection from multispectral imagery
- Identified and resolved architectural mismatches between segmentation and classification objectives
- **Stack:** Python · TensorFlow · NumPy · Remote Sensing Datasets · GIS Tools
